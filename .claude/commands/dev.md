# Start Development Server

Start the Next.js development server with hot reload enabled.

## What it does
- Runs `npm run dev`
- Starts the Next.js app on http://localhost:3000 (or next available port)
- Enables hot module replacement for instant code updates
- Watches for file changes and recompiles automatically

## Usage
```
/dev
```

## Stopping the server
Press Ctrl+C in the terminal where the server is running.

## Notes
- The dev server requires Node.js and npm to be installed
- Make sure all dependencies are installed with `npm install` first
- The .env.local file can be used to set environment variables like ANTHROPIC_API_KEY
